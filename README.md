# Madhav Ecommerce Sales Dashboard

This project is a Power BI dashboard created to analyze and visualize the sales performance of Madhav Ecommerce. The main aim of the dashboard is to understand sales patterns, customer behavior, and product performance through clear and interactive visualizations.

## About the Project

In this project, I used Power BI to transform raw ecommerce sales data into meaningful insights. The dashboard provides an overview of important business metrics such as total sales, profit, quantity sold, and category-wise performance.

## Tools Used

* Power BI
* Data Visualization
* Basic Data Analysis

## Key Insights from the Dashboard

* Overall sales and profit performance
* Category-wise and sub-category sales analysis
* State-wise sales distribution
* Monthly profit trends
* Top performing products and categories
